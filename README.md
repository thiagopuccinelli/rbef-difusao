# rbef-difusao
Este repositório contém os arquivos de entrada utilizados nas simulações de dinâmica molecular apresentadas no trabalho “Difusão e adsorção em meios desordenados: uma proposta computacional para o ensino de física estatística”. São disponibilizados os scripts de simulação, arquivos de configuração inicial e rotinas de análise empregadas na obtenção das propriedades estruturais e dinâmicas discutidas no artigo, incluindo coeficientes de difusão, funções de distribuição radial e potenciais de força média. O objetivo do repositório é garantir a reprodutibilidade dos resultados e fornecer uma base acessível para fins didáticos e de exploração computacional em física estatística e dinâmica molecular.
